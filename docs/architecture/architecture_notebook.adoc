. CSS / Javascript / HTML

=== Serverseitige Programmierung mit Python

Es wurde sich in diesem Fall für Python entschieden, da durch das Framework 'Flask' die Serverseitige Programmierung sehr übersichtlich gehalten wird. Es ist sehr einfach möglich, URL-Pfade anzupassen oder zu erweitern. Des Weiteren bestand im Bezug auf Flask innerhalb des Teams vereinzelt Vorerfahrung.
